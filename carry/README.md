# xn/carry — 股指期货吃贴水 (basis carry)

研究问题：**IC/IF/IH/IM 的贴水是多大、多稳、能不能吃、什么时候更肥？**
（文献：Koijen-Moskowitz-Pedersen-Vrugt 2018 *Carry* JFE；De Roon-Nijman-Veld 2000 对冲压力）

## 数据与方法（关键：不需要现货行情）

- 表：`dfs://future_day_kline/DayKLine`（月份合约日线 close/settlement，IF 2010-04 起、
  IC/IH 2015-04 起、IM 2022-07 起）+ `DailyInstrumentInfo`（每合约最后交易日）。
- ⚠️ **`ICInd` 等 `{P}Ind` 不是现货指数**（到期日不收敛，2024-06-21 差 1.4%，像持仓加权
  期货合成指数）；仓库里也没有 000905 等现货日线（hft_stock_ts 只有个股 + 3只ETF一年）。
  这台机器外网不通，拉不了公开数据。
- 解法两条腿，全部纯期货：
  1. **期限结构 slope carry**（KMPV 式）：`(F近−F远)/F远 × 365/Δ天 ×100`，日频。
  2. **交割结算价锚定的实现 carry**：CFFEX 交割结算价 = 现货指数到期日最后2小时均价，
     是精确现货锚。滚动周期：上一合约到期日按收盘买入次月合约，持有至其交割。
     实现 carry = (DS₁/F₀ − 1) − (DS₁/DS₀ − 1)。分红自动包含（两腿都是价格指数口径）。

## 结果（2015-04…2026-07；IF 含 2010 起）

| | IC | IF | IH | IM |
|---|---|---|---|---|
| 实现 carry 年化 | **+11.9%** | +2.9% | +4.0% | **+10.0%** |
| t 值（月频） | 9.9 | 3.0 | 4.5 | 7.7 |
| 年化 Sharpe（对指数超额） | 3.0 | 0.75 | 1.35 | 4.0 |

1. **实现 carry ≈ −入场基差，相关 −0.99~−1.00 = 恒等式**。持有到交割时收益入场即锁定
   （DS₁/F₀−DS₁/DS₀ ≈ −基差×(1+现货收益)）。吃贴水不是预测，是**读屏幕上的期限溢价**；
   风险只在路径（保证金/浮亏）和提前平仓（基差可能走阔）。
2. **历史形态（fig201）**：IF 2010-14 长期**升水**（负carry，散户投机压力）；2015-16 交易
   限制后贴水爆炸（年化 30-50%）；2017 后 IC 稳定 5-15%，IM 上市即深贴水；2023 一度收窄
   （雪球敲入平仓），2025-26 回到 10%+。**贴水=对冲需求的价格**，随中性/雪球产品规模起落。
3. **分红季节性（fig203）**：5-8月到期窗口 carry 显著更高，那部分是分红补偿
   （价格指数口径下期现双方都不含分红，故对"替代持指数"的用户仍是真实超额）。
4. **深贴水入场更肥且现货并不因此更差到抵消（fig204 + 时机表）**：入场基差深于扩张中位数时
   IC +1.64%/月 vs 浅时 +0.54%/月。深贴水后现货确实略弱（IC −0.46% vs +0.93%），
   即贴水部分是公平的弱市补偿，但**期货多头总收益在深/浅入场后几乎一样**
   （IC +1.32% vs +1.14%）——对"用期货替代指数敞口"的持有者，贴水深浅都值得吃，深更肥。
