# 股指期货分钟内动量/反转研究 · 完整报告

**标的：** 连续合约 IC（中证500）、IF（沪深300）、IH（上证50）、IM（中证1000）
**数据：** DolphinDB `hft_future_ts`，500ms 快照中间价，**2020-01 至 2026-05，共 76 个月**
**时段：** 上午 09:30–11:30，下午 13:00–15:00（约 240 分钟/天）

**核心结论：** 分钟内存在**真实、跨月稳定、样本外稳健**的微观结构（对上一分钟方向的均值回复 + 弱尾盘动量）。但其幅度（0.05–0.3 指数点）**普遍比实测买卖价差（0.5–1.0 点）小一个量级**，因此扣除真实成本后**没有任何时段、任何方向、任何精细度可以盈利**——这是穷尽证明，不是抽样。**有物理，无套利。**

---

## 1. 目标
检验一个固定、简单的分钟内择时规则能否在流动性极好的股指期货上产生**扣成本后的真实收益**；若不能，则用严格方法（跨月稳定性、样本外纪律、真实成本）说清楚**为什么不能**。

## 2. 策略（全程固定）
每分钟 M，按**上一分钟**的方向开仓，持有整一分钟：
- 信号 `d_M = sign(close(M−1) − close(M−2)) ∈ {−1,+1}`
- 盈亏 `pnl_M = d_M × (close(M) − open(M))`
- `>0` = 延续（动量赚），`<0` = 反转（被上一分钟方向打脸）。仅当 M−1、M−2 连续且该分钟 ≥2 个 tick 时计入。

## 3. 核心定义（公式）
某分钟逐tick中间价 `P_1 … P_L`（L≈90–120）：开盘 `P_1`、收盘 `P_L`、位移 `X_k = P_k − P_1`。
- 每分钟盈亏：`pnl_M = d_M(P_L − P_1)`
- 曲线按**真实tick序号**对齐（不做时间归一化），跨日逐tick求平均。
- 跨月相关：两月曲线沿tick轴的 Pearson 相关。
- 方差比（一/二阶矩互证）：`VR(k) = Var(X_k)/[(k−1)·Var(X_2)]`，`<1` 均值回复，`=1` 随机游走，`>1` 动量。
- 可交易性：`净 = E[pnl] − 成本`，`成本 ≈ E[ask − bid]`（市价单，一笔来回付一个完整价差）。

## 4. 完整数值例子 — IC，2023-03-01，09:37
| 量 | 计算 | 值 |
|---|---|---|
| 信号 | `sign(6342.40 − 6336.70)` | `d = +1`（做多） |
| 开盘 / 收盘 | `6342.80 / 6337.20` | — |
| **盈亏** | `+1 × (6337.20 − 6342.80)` | **−5.60 点（反转）** |

做多（上一分钟涨），但价格跌破开盘 5.6 点 → 动量被打脸。把同一分钟跨多日的此量平均，就得到"先降后部分回升"的曲线。

## 5. 发现一 — 方向结构（框架决定一切）
- **买入持有** `(P_L−P_1)`（永远做多）：均值≈0、跨月相关≈0 → 像纯噪声。
- **动量** `d·(P_L−P_1)`：平均曲线**前1/3分钟下探到 −0.15…−0.25 点（谷底~tick40）再部分回升** = 对上一分钟方向的短期**均值回复**。
- **为何必须乘信号：** 反转是**以上一分钟方向为条件**的（涨后跌、跌后涨）。买入持有把两种相反情形平均到≈0；乘 `d` 后两者同号、得以存活。相关性从≈0 跳到 **+0.44~0.65（原始tick）**。
- **二阶矩印证：** 方差比 `VR≈0.7<1`，独立证实分钟内轻度均值回复。

## 6. 发现二 — 跨月稳定是真的，不是统计假象
累积曲线相关会虚高（两条平滑积分序列即使独立也相关），故做差分检验：
- **增量（差分）相关跨月仍 +0.15~0.19**（零假设≈0，p95≤0.005）→ 逐tick形状真共享，免疫累积陷阱。
- 约 **95% 的月份**呈相同"先降再升"形状。

## 7. 发现三 — 时段结构（过76月 + 样本外验证）
逐分钟盈亏聚合到15分钟时段，得到稳定的时段地图：
- **反转**（动量亏）：盘中 10:00–11:00、13:15–13:30、14:15。
- **延续**（动量赚）：午休前 11:15–11:30、午后开盘 13:00、冲收盘 14:45–15:00。
- 方向延续到全76月，但仅为 **~60–75% 月份的倾向**（平静月子样本里曾~100%）——真实但中等。

## 8. 可交易性检验（全部样本外 + 真实成本，本项目核心）

| 检验 | 方法 | 结果 |
|---|---|---|
| 全天每分钟做 | ~230笔/天 | **毛利就为负**（IC −0.062 / IF −0.057 / IH −0.037 / IM −0.168 点/笔）→ 分钟内净反转，动量扣成本前已亏（图40） |
| 选最优分钟 | 挑样本内显著分钟 | IS↔OOS相关≈0，t=−6的分钟样本外**反号** → 过拟合（图32） |
| 波幅条件化 | 按上一分钟波幅分桶 | 样本外**失效**（t<1，图33） |
| 15分钟时段筛 | 聚合到时段 | 浮现真实时段结构 ✓（图34–36） |
| 整段持有1笔 | 11:21进/11:30出 | 上一分钟方向**预测不了**9分钟整段（t<1） → 不行 |
| 最强段深挖 | 11:21–11:29 每分钟动量 | 毛 **+0.22(IC)/+0.28(IM)** 点/笔，74–76%正月，**假设**0.1成本时净值稳升 ✓ |
| **做实：实测价差** | 测 11:21–11:29 真实 `ask−bid` | IC/IM ≈ **1.0点**、IF/IH ≈ 0.5点 = **毛利的2–5倍** → 净 −0.4…−0.8/笔，76月净值崩到 **≈ −200万元/手**（图38） |
| 穷尽扫描 | 全16时段×4合约 vs 各自真实价差 | **65/65 全在盈亏平衡线下，净>0 个数=0**（图39） |
| 四个session边缘 | 开盘/收盘各窗口精确到分钟 | edge 与价差**同涨同跌** —— 开盘反转最大(~0.5) 但价差也最宽(~1.07)；16格全部净亏（图41、图42） |
| 开盘动量·剔反转 | 只保留 09:38–39/09:44 的动量分钟 | 最优净 −0.44(IF)~−0.73(IM)；开盘是**价差最宽**时段，大反转也付不起 → 全天最差（图43） |
| 逐合约潜力排名 | 每窗口按 `\|edge\|−价差` 排 | 潜力集中在**下午14:00–14:45**；**IM 14:15反向 最接近盈亏平衡(净 −0.03，83%月份同号)**，但仍为负；IH 全无潜力（图44） |
| 纯方向赌注 | "专心做空14:15→14:30"，无信号 | **胜率≈48–50%（抛硬币）**，区间漂移≈0（甚至微涨），净 −0.55…−2.23 → 证实**波动≠优势**：无方向预测时，方向赌注=抛硬币−价差 |
| **tick级反转** | 开盘进，固定tick退出（扫20/42/60/90/收盘） | 在反转谷底（**tick~42，约20秒**）平仓比收盘多抓 **1.5–2.4倍**毛利（IC +0.147 vs +0.062）—— 但最优毛利(0.07–0.25)仍**比价差小4–7倍** → 任何退出点净亏（图45） |

### 决定性算术
`IC 11:21–11:29：毛 +0.225 − 价差 1.023 = 净 −0.798 点/笔。`

### 结构性原因（关键洞见）
1分钟价格移动本就小（edge 0.05–0.3），而每笔都付一次价差（0.5–1.0）→ **edge 天生比成本小一个量级**。两条互相强化的事实让它无解：
1. **edge 与价差同涨同跌。** 高edge时段（开盘，反转~0.5）恰是宽价差时段（~1.07），二者绑定 → 全天**没有"高edge+低成本"的甜区**（图41、44）。
2. **精细化也撞墙。** 各种粒度都试过——分钟收开、15分钟块、逐分钟、直到 **tick级最优退出（图45）**。每细一层都多榨一点毛利（择时直觉对），但最优（tick42退出，0.07–0.25）仍比价差小4–7倍。反转在每个尺度都真实，只是比"碰一次市场"的成本小。

唯一能让"买在某tick卖在某tick"翻正的，是**挂限价单做市（赚价差，而非付价差）**——执行/做市能力问题，不在这份数据范围内。

## 9. 最终结论
1. **存在真实微观结构：** 对上一分钟方向的条件均值回复（方差比<1印证）+ 弱尾盘动量；跨月稳定、样本外稳健、增量检验确认。
2. **但全都不可交易：** 每个方向性 edge 都深埋在买卖价差里。这是**穷尽**结论——覆盖所有时段（65/65失败）、两个方向（动量与反向）、四个合约、以及**直到tick级最优退出的每一种粒度**。
3. **纯方向赌注没有优势：** "专心做空"某时段 = 抛硬币（≈50%胜率、≈0漂移）减价差 → 必亏。波动是方差不是期望；没有方向预测就赚不到波动。
4. **"找分钟内可交易edge"这条线在数据层面已封死。** 有物理，无套利。
5. **唯一未封的门不在数据里：** 用限价单/做市被动成交**赚**价差而非付价差——执行/提供流动性的能力问题，超出本数据集。

## 10. 方法论收获（可迁移）
- 必须乘信号 `d` 才能看见**条件**效应；无条件平均会把它藏掉。
- 累积曲线的相关会虚高——务必用**增量 + 打乱零假设**确认。
- 可交易性必须用**实测价差**，绝不用假设成本：0.1 vs 1.0 就是"盈利"与"−200万"之差。
- 每个"样本内看着能赚"的候选都必须过**严格样本外**：最强的样本内分钟（t=−6）样本外反号。
- 警惕regime混合：绝不跨平静月与危机月池化方差（曾把方差比灌到4）；先月内归一、再取中位。
