# 分钟内动量/反转研究 · 阶段性总结 (`xn/intraminute`)

**标的**：中国股指期货连续合约 IC(中证500) / IF(沪深300) / IH(上证50) / IM(中证1000)
**数据**：DolphinDB `hft_future_ts`，500ms 快照中间价，2020-01 ~ 2026-05 共 76 个月
**一句话结论**：分钟内存在**真实、跨76月稳定、样本外也成立**的微观结构（对上一分钟方向的均值回复 + 尾盘动量），但其幅度（0.05–0.3 指数点）**普遍小于真实买卖价差（0.5–1.0 点）一个量级**，穷尽扫描后**无任何时段可在真实成本下盈利**。有物理，无套利。

---

## 1. 基本构造（Equations）

### 1.1 原料
某分钟 $M$，逐tick中间价 $P_1, P_2, \dots, P_L$（$L\approx$90–120），
$$\text{mid}=\tfrac{bid+ask}{2},\quad \text{开盘}=P_1,\ \text{收盘}=P_L$$
分钟内相对开盘位移：$\;X_k = P_k - P_1\;$（故 $X_1=0$）。

### 1.2 动量信号（策略核心，固定不变）
$$d_M = \operatorname{sign}\big(\text{close}(M{-}1) - \text{close}(M{-}2)\big)\in\{-1,+1\}$$
上一分钟涨→$d=+1$（做多）；跌→$d=-1$（做空）。要求 $M{-}1,M{-}2$ 连续，否则该分钟作废。

### 1.3 每分钟动量 P&L
$$\boxed{\ \text{pnl}_M = d_M\times\big(\text{close}(M)-\text{open}(M)\big)=d_M\,(P_L-P_1)\ }$$
$\text{pnl}>0$=动量延续（信号方向继续走）；$\text{pnl}<0$=反转（被上一分钟方向打脸）。

### 1.4 分钟内曲线（原始tick，未归一化）
对同一 minute-of-day，把每天的逐tick位移 $X_k=P_k-P_1$ 按**真实tick序号**对齐（不做时间插值），跨所有交易日**逐tick求平均** → 一条曲线/分钟。

### 1.5 跨月相关（结构是否重复）
每月得到一条曲线 $C_m(\cdot)$，两月份沿tick轴做 Pearson：
$$\rho(a,b)=\frac{\sum_k (C_a(k)-\bar C_a)(C_b(k)-\bar C_b)}{\sqrt{\sum_k(C_a(k)-\bar C_a)^2}\sqrt{\sum_k(C_b(k)-\bar C_b)^2}}$$

### 1.6 方差比（一/二阶矩互证）
$$VR(k)=\frac{\operatorname{Var}(X_k)}{(k-1)\,\operatorname{Var}(X_2)}\quad\begin{cases}<1 & \text{均值回复}\\=1 & \text{随机游走}\\>1 & \text{动量}\end{cases}$$

### 1.7 可交易性（最终判据）
$$\text{净/笔}=\underbrace{\text{毛利/笔}}_{\mathbb{E}[\text{pnl}_M]}-\underbrace{\text{成本/笔}}_{\approx\,\text{买卖价差}\,\mathbb{E}[ask-bid]}$$

## 2. 一个完整数值例子（IC, 2023-03-01, 分钟 09:37）

| 步骤 | 计算 | 结果 |
|---|---|---|
| 信号 | $d=\operatorname{sign}(6342.40-6336.70)=\operatorname{sign}(+5.70)$ | $d=+1$（做多） |
| 开盘 | $P_1=$ open(09:37) | 6342.80 |
| 收盘 | $P_L=$ close(09:37) | 6337.20 |
| **P&L** | $\text{pnl}=+1\times(6337.20-6342.80)$ | **−5.60 点（反转）** |

→ 信号让我们做多，但这一分钟价格跌破开盘 5.6 点 → 动量被打脸 = 反转。把同一分钟跨多日的逐tick位移平均，就得到 09:37 那条"先降后部分回升"的曲线。

---

## 3. 主要发现（按层次）

### 3.1 一阶矩（方向）：buy-and-hold 是噪声，动量框架才显形
- **永远做多** $(P_L-P_1)$：均值≈0，跨月相关≈0（看着像噪声）。
- **乘上信号** $d\,(P_L-P_1)$：平均曲线**前1/3分钟下探到 −0.15~−0.25 点（谷底~tick40）再部分回升** = **对上一分钟方向的短期均值回复**。
- **为什么必须乘 $d$**：反转是*以上一分钟方向为条件*的（涨后跌、跌后涨）。buy-and-hold 把两种相反情形平均到≈0；乘 $d$ 后两者同号，跨月相关从 ≈0 跳到 **+0.44~0.65（原始tick）**。
- **不是累积假象**：对曲线做差分（增量）后跨月相关仍 **+0.15~0.19**（零假设≈0）→ 形状真共享。
- **形状稳定**：95% 的月份是"先降再升"（图26）。

### 3.2 二阶矩（佐证，后续按需删除）
方差比 $VR\approx0.7<1$（谷底~tick55）→ 亚扩散 = 轻度均值回复。**一阶矩(曲线下探)与二阶矩(VR<1)两条独立路径得出同一结论**。

### 3.3 跨年月：方向稳定但中等
全76月验证：盘中 10:00–11:00、13:15–13:30、14:15 **反转**；11:15–11:30、14:45 **延续**——方向延续到全历史，但一致性从平静月的~100%降到 **~60–75% 的月份**（中等倾向，非铁律）。

## 4. 可交易性检验（全部样本外 + 真实成本）

样本内 IS = 6个平静月(2023-05…2025-06)，样本外 OOS = 6个平静月(2023-10…2025-10)。

| 尝试 | 方法 | 结果 |
|---|---|---|
| 全天每分钟做 | 开盘30min每分钟一笔 | 毛≈0、胜率49%，~28笔/天×成本 → **净大亏** |
| 选最优分钟 | 挑IS显著分钟 | IS↔OOS相关≈0，t=−6的分钟样本外**反号** → 过拟合 |
| 波幅条件化 | 按\|上一分钟动\|分桶 | IS形态样本外**失效**(t<1) |
| **时段筛** | 15分钟时段聚合 | 浮现真实**时段结构**（见3.3）✓ |
| **最强段深挖** | 11:21–11:29 每分钟做动量 | 毛 **+0.22(IC)/+0.28(IM) 点/笔**，74–76%正月，假设0.1成本时净值稳升 ✓ |
| **做实：实测价差** | 测11:21–11:29真实 $ask-bid$ | **IC/IM ≈1.0点、IF/IH≈0.5点 = 毛利的2–5倍** → 净 −0.4~−0.8/笔，76月崩到 **−200万元/手** |
| **穷尽扫描** | 全16时段×4合约 vs各自真实价差 | **65/65 全在盈亏平衡线下方，净>0 个数=0** |

### 关键公式回到判据
- IC 11:21–11:29：毛 $+0.225$ − 价差 $1.023$ = **净 $-0.798$ 点/笔**。
- 结构性原因：1分钟价格移动本就小（edge 0.05–0.3），而每笔都付一次价差（0.5–1.0）→ **edge 天生小价差一个量级**。

---

## 5. 最终结论

1. **存在真实微观结构**：对上一分钟方向的均值回复（盘中/午后）+ 尾盘动量延续；跨76月稳定、样本外成立、被方差比独立佐证。
2. **但不可交易**：所有方向性 edge（0.05–0.3 点）都**深埋在买卖价差（0.5–1.0 点）里**。这是穷尽扫描的结论（65/65 失败），不是抽样猜测。
3. **方法论收获**：①必须乘信号 $d$ 才能看见条件反转；②跨月相关要用**增量**排除累积假象；③可交易性必须用**实测价差**而非假设成本——0.1 vs 1.0 的差别就是"赚钱"和"破产"的差别；④每一个"看着能赚"的信号都被严格样本外+真实成本否决。
4. **唯一未否的出路**（不在数据内）：把单笔执行成本从~1.0点压到~0.1点（极致做市/被动成交）——这是执行能力问题，非信号问题。
