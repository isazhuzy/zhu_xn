## 0. 一句话总览

> 把每个 tick 按某个「订单流因子」排序，累计它之后的未来价格变动——单点看不见的微弱预测力
> 在累计曲线的**形状**里现形（斜率=条件均值收益，谷底=多空阈值）。把「回看多久算因子」和
> 「持有多久算收益」两个时间窗口放开成二维平面后发现：**同一个因子在不同窗口下会从动量翻成
> 反转**，且 VOI / MPB / OIR 三个因子各有完全不同的窗口性格。

## 1. 数据与预处理

**数据源**：DolphinDB `dfs://hft_future_ts` 表 `TickPartitioned`，500ms 快照（每分钟约 120 tick）。
字段：`m_nDatetime, m_nPrice(最新价), m_nBidPrice/Volume, m_nAskPrice/Volume, m_iVolume(区间量),
m_iTurnover(区间额)`。连接见 `xn/prediction/ddb_config.py`。

**已知数据坑**（记忆库 `ddb-tick-data-quirks`）：
- 节奏 500ms / 每分钟约120 tick；2024-02 有重复行（需 `drop_duplicates("ts")`）；
- 2022-12 tick 异常密集、2023-07 部分缺失；**IM 合约数据自 2022-07 才有**（历史最短，最噪）；
- 涨跌停时 ask=0 → mid 会算成半价（需过滤）；`m_iVolume` 是**区间增量**、AccVol 才是累计。

**清洗步骤**（`prep()`，三份脚本一致）：
1. `drop_duplicates("ts")` 去重、按时间排序；
2. 保留 `pb>0 且 pa>pb 且 px>0`（合法双边报价）；
3. 交易时段过滤：09:30–11:30 与 13:00–15:00（`tod` 分钟数区间）；
4. 价差 `spr = round((pa−pb)/TICK)` 保留 1–50 tick（去异常）；
5. **gid = 日期×上下午**（`normalize()*2 + (下午?1:0)`）——所有滚动/前向计算都**在 gid 内**做，
   绝不跨午休/隔夜。

## 2. 三个因子（每 tick 计算）

| 因子 | 类型 | 公式 | 备注 |
|---|---|---|---|
| **VOI** | 流量(flow) | ΔV_bid − ΔV_ask，带 down/same/up 规则 | Shen(2015)；整数手；每场首 tick 无前值=NaN |
| **OIR** | 盘口水平(level) | (qb−qa)/(qb+qa) ∈ [−1,+1] | 无需前值；QI=(OIR+1)/2 |
| **MPB** | 成交侧压力 | 区间成交均价 − 前两个 mid 的均值（tick 单位） | vol==0 时 ffill 上一笔成交价；首 tick=NaN |

**VOI 的 down/same/up 规则**（`add_voi`）：
```
ΔV_bid = 0            若 bid价下移
       = qb − qb(-1)  若 bid价不变
       = qb           若 bid价上移
ΔV_ask 对称（ask价上移→0，不变→差分，下移→qa）；VOI = ΔV_bid − ΔV_ask
```
**MPB**（`add_mpb`）：`成交均价_tk = amt/(vol×合约乘数)/TICK`（vol==0 用 ffill），
`mpb = 成交均价 − (mid_tk + mid_tk(-1))/2`。合约乘数 `MULT` 见 `lob_common.py`。

**符号约定（三因子统一）**：**正值=买压，负值=卖压**。
- VOI>0：买边量增快于卖边；OIR>0：买挂单更多；MPB>0：成交印在中价之上（买方主动扫）。

## 3. 方法论 A — 排序累计诊断（sorted-cumsum，最初的方法）

### 3.1 核心三行
```python
order = np.argsort(a["voi"], kind="stable")   # ① 按因子值从小到大排（返回下标）
cum   = np.cumsum(a["dy20"][order])           # ② 按此顺序累计未来收益
q     = rank / n                              # ③ x轴=百分位（经验CDF）
```
- **outcome**：`dy_k(t) = P(t+k) − P(t)`（tick 单位），`shift(-k)` 在 gid 内做，|dy|>100 丢弃。
- **downsample**：全 n 行累计，只存 4000 个等距名次（`np.linspace`）——视觉不变、文件小万倍。
- 内存：code 外层 / month 内层循环（排序一次只需一个合约），只留必要列，float32。

### 3.2 为什么有用（统计核心）
单 tick 的 dy 信噪比 ≈ 0.1（±几 tick 噪声包着 ±0.3 tick 均值），散点图什么都看不见。但：
- **噪声正负相抵，n 步后只游走到 √n 量级；偏差（真信号）每步同向，累积成 n 量级。**
- 100 万 tick 后：0.2 tick 的偏差累成 20 万，噪声只到 ±3000。**cumsum 用 √n 倍放大系统性倾向。**

### 3.3 排序后 cumsum 的额外含义
> **曲线任一点的斜率 = 该因子水平的平均未来收益 E[dy | factor]。**
- 左段下行=负因子后跌，右段上行=正因子后涨；**对勾形(✓)=单调有预测力**。
- **最低点 = 条件均值换号处 = 天然多空阈值**（三因子都在 ≈0）。
- 曲率=尾部是增强还是饱和。回归只给一个斜率+R²；排序 cumsum 给出整条反应函数，不假设线性、自带 √n 降噪。

### 3.4 读图两眼法 + 关键警句
1. **谷底深度 ÷ n = 每笔毛收益**（信号强度）。
2. **终点 vs 0 = 样本整体漂移 ×(约 k 倍窗口重叠)，与因子无关**——三因子终点几乎一样（IC 都约+8万），
   排序改变不了总和。要去漂移：累计前对 dy 去均值。
- ⚠️ **相邻 tick 的 k 窗口重叠（共享 k−1 个快照），纵轴数值被膨胀约 k 倍；形状和每笔深度才诚实。全部毛收益、sub-spread。**

### 3.5 原始结果（cumsum 深度，k=20=持有10s，tick/笔）
- **VOI**：四合约都是干净深 V，全分布单调（非尾部驱动）。每笔深度 IM .46 / IC .43 / IF .24 / IH .17
  ——小盘约 2× 信号。k=1 只captures ~40%，**k=20 ≈ k=120**（约10s 冲击完成）。
- **OIR**：圆 U，比 VOI **弱约 5×**（IC k=20 深度 .073）；约 5s 饱和。盘口的*倾斜*有信息、*变化*才是真信号。
- **MPB**：对称深 V，每笔深度介于两者之间（IM .34/IC .29/IF .16/IH .11，约 VOI 的 ⅔）。
  **比 VOI 快**：k=1 已 captures ~⅔（VOI 只 ~40%）——成交侧信息几乎瞬间被定价。
  因子排序：**流量(VOI) > 成交侧(MPB) > 水平(OIR)**，跨合约都 IM>IC>IF>IH。

## 4. 方法论 B — 极端尾部的单笔收益（tail buckets）

某段样本的单笔均值 = 曲线在那段的**斜率**：
```
mean_dy(q1→q2) = [cum(q2) − cum(q1)] / [(q2−q1)×n]
```
**桶定义**（嵌套的分位尾部，6 桶）：
| 桶 | 定义 | 含义 |
|---|---|---|
| 最负 0.1/1/5% | 因子 ≤ 0.1/1/5 分位 | 最强净卖压 |
| 最正 5/1/0.1% | 因子 ≥ 95/99/99.9 分位 | 最强净买压 |

嵌套：最负1% 含最负0.1%。**桶均值不受窗口重叠膨胀**（每 tick 只贡献一个 dy），比 cumsum 纵轴更诚实。

### 4.1 结果（原始 §五，瞬时因子 / k=20 / VOL/OIR 用最新价、MPB 用中间价）
**三种「尾部性格」**：
- **VOI = 尾部动量型**：5%→1%→0.1% 一路变陡（IC 1.9→2.7→4.1），越极端冲击越狠。全项目唯一明显 >
  一个往返价差的毛收益——但 0.1% ≈ 每天每边仅~25 次，且都在行情最猛、最难成交时。
- **MPB = 饱和型**：印在买/卖价这个「消息」一旦送达，更极端只是噪声（S 型平顶）。
- **OIR = 尾部反转型（IM）**：盘口压到最一边反而反弹——挂单耗尽/涨跌停微观结构；回归会把它平均进斜率、永远看不见。

## 5. 方法论 C — Experiment C：回看 × 持有 窗口平面（本轮主线）

原始 cumsum/尾部固定了**两个**时间窗口。Experiment C 把它们放开成二维平面。

### 5.1 两个窗口 + 关键计算事实
- **回看 J**（J tick = J×0.5s）：`factor_J(t) = Σ_{i=t−J+1..t} factor(i)`，在 gid 内滚动和，J=1=瞬时。
  > **关键**：我们按**分位排序**（纯 ranking），且每个保留 tick 恰好求和 J 项 → 滚动**和** ≡ 滚动**均值**
  > （均值=和/J，J 恒定）→ 同样排序 → 同样的桶 → 同样的数。所以「和 vs 均值」在此无关。
- **持有 k**（k tick = k×0.5s）：前向收益，`shift(-k)` 在 gid 内。

### 5.2 价格基准规则（按因子分）
- **VOI、OIR → 最新价** `dy_k = last(t+k) − last(t)`；
- **MPB → 中间价** `dy_k = mid(t+k) − mid(t)`（**强制**）。
- **为什么**（bounce 教训，本系列最重要的一课）：MPB 就是「最后一笔成交相对中价的位置」，若用最新价测收益，
  买卖价 bounce 是与信号等大的反向项、恰好抵消——用 `PRICE=last` 时 MPB 的对勾**完全消失**（谷底跑到 q=1.0）。
  > **规则：outcome 必须与 factor 测量独立。成交型因子 → 用报价型 outcome。**
  > VOI/OIR 因为是报价构建的，用最新价只加噪声、不加相关偏差，所以「幸存」。

### 5.3 统一时间刻度
两个轴统一为 **0.5s / 2.5s / 10s / 30s / 60s / 120s / 300s**（tick [1,5,20,60,120,240,600]），
避免「回看有 2.5s、持有没有」的不一致。

### 5.4 动量/反转的判定（去漂移分解）
对每个 (因子, 合约, J, k)，取两端 0.1%（或 1%）桶均值 P(最正)、N(最负)：
```
S = (P − N)/2   # 去漂移预测信号：S>0 动量（买→涨/卖→跌）；S<0 反转（买→跌/卖→涨）
A = (P + N)/2   # 公共分量 = 样本漂移 + 真实买卖不对称
```
- **|S| = 信号强度**（挑「最强窗口」用 1% 尾的 |S| 跨合约平均，稳）。
- **漂移很小**：实测 A 在 J=1 时 <0.1 tick（到60s）、<0.4 tick（到300s）→ 漂移**不是**主因，尾部的买卖不对称是真微观结构。

### 5.5 最强窗口选择（Table 3）
用 **1% 尾** 的 `|S| = mean_合约|最正1%−最负1%|/2` 打分（1% 尾样本多、稳），
并**排除 300s 重叠噪声边**（300s 窗口相邻样本重叠 599/600，有效样本极少、会伪造尖峰）。

## 6. 核心结果 — 三个因子三种窗口性格

（全样本，tick/笔毛收益，VOI/OIR 最新价、MPB 中间价）

| 因子 | 类型 | 短回看/短持有 | 如何翻成反转 | **翻转由哪个窗口驱动** |
|---|---|---|---|---|
| **VOI** | 流量 | 动量，~10s 达峰 | 回看**和**持有都拉长 → 反转 | **回看**（对角线翻转）|
| **MPB** | 成交侧 | 动量，~1 tick，10s 内消失 | **不翻**，只衰减到 0，无反转翼 | （无）|
| **OIR** | 盘口水平 | 仅 <1s 动量 | ~10s 起翻反转并随持有增强 | **持有**（回看几乎不影响）|

### 6.1 各因子最强窗口下的桶×合约（1% 尾）

### 6.2 VOI 对角翻转（招牌形态，IC/IM 最清晰）
VOI 最负1% 卖压桶，随回看↓从动量（负）翻成反转（正）：
- IC 卖压桶：回看0.5s 行 −2.51（10s持有，价继续跌=动量）→ 回看60s 行 +2.55（300s持有，价反弹=反转）。
- 最极端 0.1% 卖压桶在（回看60s/持有300s）达 **+7.38 tick/笔——全研究最大数**（累积卖压后大反弹）。
- **不对称**：重度累积**卖压**猛烈反弹，重度**买压**只是衰减到 0（对称的话买压桶该 ≈−7，实际 ≈0）。
- 小盘（IC/IM）强，大盘 **IH 几乎不翻**。

### 6.3 各因子「代表桶/代表合约」筛选结论
- 代表桶标准（用户选）：**双边 1%**（最负1%+最正1%）——够尖锐（边际是5%的~1.4倍）、~220次/日/侧不噪、保留双边不对称。
  （备选：0.1% 最尖锐但~22次/日最噪；5% 最稳|均值|×√n与总PnL最大但单笔最小。）
- 代表合约（招牌形态强度）：**VOI→IM**（翻转幅度3.79，与IC并列；用户偏好，动量峰~2.66、0.1%可达+3.75~4）、
  **OIR→IM**（反转强度1.18 远超 IF0.75/IC0.70）、**MPB→IC**（快动量0.98、衰减最单调干净；IM 1.21更大但更噪、样本自2022才有）。

## 7. 显著性 / 统计口径

- **样本量**：0.1% 桶 ≈ 22 次/日/侧、1% ≈ 220、5% ≈ 1100（约1500交易日）。
  单笔边际随分位变深单调↑，但样本/可信度/总PnL 反向↓。1% 是拐点。
- **可信度指标**：`|均值|×√n`（类 t 值）在 **5% 桶最大**；`|均值|×n`（总毛PnL/容量）也在 5% 最大。
- **漂移控制**：`A=(P+N)/2` 度量漂移+不对称，实测 <0.4 tick（300s）→ 结论对漂移稳健。
- **重叠**：长窗口（回看/持有 120s、300s）相邻样本重叠严重、有效独立样本远少于 tick 数 → 底/右角偏噪，只看趋势。
- **成本**：全部**毛收益、sub-spread**；1–5 分钟持有有真实成本/风险。极端桶都发生在行情最猛、成交最难时。

## 8. 两个测量层教训

### 8.1 bounce（§5.2 已述）
outcome 必须与 factor 测量独立；成交型因子配报价型 outcome。否则 bid-ask bounce 与信号等大反向、抵消。

### 8.2 lattice binning（分箱锯齿，`NOTE_lattice_binning.md`）
> **分箱任何价格衍生量前，先问「它能取哪些值」，把箱中心放到那些格点上。**
- MPB 长在**四分之一 tick 格点**（0, ±0.25, ±0.5…）；BW=0.1 箱对不上、`np.round` 银行家舍入方向不一致
  → 大箱(低均值)与小箱(高均值)沿 x 交替 → 规律锯齿。
- 修复：`BW=0.25` + `np.floor(x/BW+0.5)*BW`（half-up）→ 平滑 S 曲线。回归端（用原始值）从不受影响。
- 推广：VOI 格点=整数手；OIR 密但堆在小整数比（0,±⅓,±½），要分箱用**分位箱**。
  **图里出现周期性抖动 ⇒ 先怀疑测量网格，不是市场。**

## 10. 诚实结论 + 后续

**结论一句话**：VOI/MPB/OIR 三个都「度量订单流压力」，但**窗口性格完全不同**，你选的 (回看,持有) 窗口
决定了你在做动量还是反转。原始 cumsum 的「瞬时因子/10s持有」窗口恰好落在 VOI/MPB 的动量峰上，
但对 OIR 那个 10s 已在反转区——同窗口读三因子会把动量信号和反转信号悄悄混在一起。

**后续候选**：
1. 把 VOI 卖侧反转（+7 tick 角）按年份/regime 拆，确认不是 2020-COVID / 2024-波动尖峰集中。
2. 对每个前向列减去无条件均值，出纯 `S` 图（比原始尾部更干净）。
3. 用双边1% 代表桶做一个可交易性回测（含成本、独立样本修正）。
4. `voi/spr`（价差归一的 VOI，model B 实际回归量）跑同样平面。

**永远的备注**：全部毛收益、sub-spread；纵轴/长窗口受重叠膨胀；看形状、看谷底、看斜率，不抠绝对数。
