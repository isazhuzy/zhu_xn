# experiment/ — 跨合约 × 分年度 检验（方向1+2）

> **目的**：把 `../future/` 在 **IM、4 年、池化**上得到的"价格脉冲反转"结论，拿到
> **IC/IF/IH/IM 四个合约、IC/IF/IH 全 2015–2026、按年分组**去验证——这是
> **样本外 / 跨合约 / 分年度**的稳健性检验（呼应 ticker 那次"开盘反转 OOS 蒸发"的教训）。
>
> **一句话结论**：`future/` 那个"大脉冲 → 超价差反转"的亮点 **被证伪**——它一半是
> **缺价坏 tick 假反转**，一半是**少数危机年专属**；逐年逐合约看，**大脉冲多数时候是趋势**。
> 真正稳健的只剩"**小脉冲微动量**"（小、在价差内、不可净交易）。

---

## 0. 做了什么

- **EXP1 价格脉冲**（同 `future/`）：脉冲 `back=mid[i]−mid[i−n]`，`|back|>k` 触发，看未来 **x=10 tick** 的符号化收益 `sign(back)×(mid[i+x]−mid[i])`，按 **(合约, 年, n, k)** 聚合。
- **网格**：n∈{1,2,3,5,10,20,30}，k∈{0,2,4,6,8,13,20,30} 价格tick，x=10。
- **两道坏 tick 防线**（关键升级）：
  1. **源头滤波** `m_nBidPrice>0 & m_nAskPrice>0`——丢掉**缺一边报价**的快照（mid 被腰斩的假跳根源）。`future/` 只做了第 2 道。
  2. **despike** 去残留单点尖峰。
- **同时测每合约每年的真实价差**（`ask−bid` 均值），作为"能不能净赚"的成本线。
- 数据：IC/IF/IH **2015–2026**，IM **2022–2026**；500ms tick；约 400 个 contract-month。

---

## 1. 三个核心结论

### ① 源头滤波揭穿："−2 点反转"大半是坏 tick
`future/` 里 IM `n=10,k=6点` 池化 = **−2.09 点**（看着超过价差、像能交易）。
这里**源头加 `bid/ask>0` 后** → 池化变成 **+0.19 点**。样本数几乎不变，说明那 −2 点是**极少数缺价坏tick的巨幅假反转**撑起来的（despike 抓不干净，源头滤波才行）。
→ **坏 tick 必须在取数源头滤，不能只靠 despike。**

### ② 样本外：大脉冲反转**逐年翻号**，集中在危机年（见 `fig_cross_byyear.png`）
大脉冲（n=10, k=6点）之后是反转(−)还是趋势(+)，**逐年在红蓝之间跳**：

| 合约 | 趋势年 / 反转年 | 反转集中在 |
|---|---|---|
| IC 中证500 | 9 / 3 | 2017-18 |
| IF 沪深300 | 8 / 4 | **2016-17（−2.65）** |
| IH 上证50 | 7 / 5 | **2016-17（−2.92）** |
| IM 中证1000 | 3 / 2 | 2022-23 |

→ "大脉冲反转"只在 **2016-17 股灾熔断余波（IF/IH）和 IM 早期** 出现；**近年（2019-2026 多数）大脉冲反而延续（趋势）**。这是 regime / 过拟合，**不是稳定结构**。

### ③ 跨合约：IF/IH 的低价差**没救场**；真正稳的是"小脉冲微动量"
- 各合约**实测价差**：IC≈**1.33**、IF≈**0.59**、IH≈**0.56**、IM≈**1.00** 点。
- 本以为 IF/IH 半价差能让反转净赚——但**反转本身在多数年份不存在**，低价差也无从越过。
- **小脉冲（k=0.8点）的趋势/延续是跨年跨合约稳健的**：

| 合约 | 小脉冲 趋势年/反转年 |
|---|---|
| IF | **12 / 0**（完美稳定） |
| IM | **5 / 0** |
| IC | 11 / 1 |
| IH | 7 / 5（最不稳的合约） |

→ **唯一稳健的边际 = "小而快的脉冲会延续"（微动量）**；但它 ≤0.1 点、在价差内，**不可净交易**。

---

## 2. 这对整个研究意味着什么

1. **方法论大胜**：先做跨合约×分年度（方向1+2），**省下了对一个假信号做净回测的功夫**——那个"超价差反转"线索若直接拿去回测，会被坏 tick + 危机年骗。
2. **`future/` 的"标志性反转格"结论已修正**（见 `../future/README.md` 顶部修正）。
3. **下一步的合理方向**：要么研究"**什么 regime 下大脉冲反转 vs 趋势**"（用波动率/价差状态把危机年和平常年分开，看反转有没有可识别开关）；要么接受"小脉冲微动量在价差内、无净edge"的负结论，转去别的信号。

## 3. 文件

| 文件 | 作用 |
|---|---|
| `exp1_cross_ddb.py` | 跨合约×分年度 EXP1 n×k（源头滤坏tick + despike）+ 测真实价差 → `exp1_cross.csv` / `exp1_cross_spread.csv` |
| `exp1_cross_plot.py` | 出图 + 文字汇总（价差/反转/胜率/是否净>0） |
| `figs/fig_cross_hardline.png` | 4 合约 年份×k 热力图（n=10）：hardline 跨年跨合约稳不稳（红趋势/蓝反转） |
| `figs/fig_cross_byyear.png` | 4 合约 大脉冲(n=10,k=6点)均值逐年柱状：**逐年翻号 = 不稳定** |
| `figs/fig_cross_spread.png` | 反转毛幅度 vs 真实价差 逐年 |
| `exp1_cross.csv` | `code,year,n,k_ticks,k_pts, s_sum,s_ss,s_n,hits(rev=signed<0)` |
| `exp1_cross_spread.csv` | `code,year, sp_sum,sp_n`（平均价差=sp_sum/sp_n） |
| `ddb_config.py` | 本地凭据（gitignored） |

**约定**：CSV 存 sum/ss/count；`mean=s_sum/s_n`，`se=sqrt(s_ss/s_n−mean²)/√n`，`t=mean/se`，`反转胜率=hits/s_n`。

## 4. 数据 / 技术注记
- 取数用 `../.venv`（dolphindb），**关沙箱**；画图用 system `python3`。
- 数据：IC/IF/IH 2015-2026、IM 2022-2026；IC/IF/IH/IM 最小变动价位均 **0.2 点**（k 跨合约可比）。
- 各合约点值：IC/IM ¥200/点，IF/IH ¥300/点。
